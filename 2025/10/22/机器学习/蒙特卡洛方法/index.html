дёүгҖҒsimulationпјҲд»ҝзңҹпјүrollout пјҲеҮәе“Ғпјү

д»ҺиҝҷдёӘж–°иҠӮзӮ№ејҖе§ӢпјҢз”ЁйҡҸжңәпјҲжҲ–еҗҜеҸ‘ејҸпјүж–№жі•вҖңзҺ©вҖқеҲ°жёёжҲҸз»“жқҹгҖӮеҸ¶еӯҗиҠӮзӮ№е°ұз®—жҳҜдёҖдёӘжёёжҲҸз»“жқҹзҡ„зҠ¶жҖҒгҖӮ
рҹ‘ү иҝҷдёҖжӯҘе°ұжҳҜеә”з”ЁвҖңи’ҷзү№еҚЎжҙӣж–№жі•вҖқ вҖ”вҖ“ з”ЁйҡҸжңәж ·жң¬зҡ„з»ҹи®Ўз»“жһңеҺ»иҝ‘дјјдёҖдёӘж•°еӯҰй—®йўҳзҡ„зңҹе®һзӯ”жЎҲгҖӮ

жіЁж„ҸпјҢдёҖдёӘйқһеёёе®№жҳ“и®©дәәиҜҜи§Јзҡ„зӮ№пјҒпјҡ

вҖӢ иҝҷйҮҢзҡ„ ж•ҙдёӘrolloutи®Ўз®—иҝҮзЁӢдёӯпјҢж¶үеҸҠеҲ°зҡ„иЎҢдёәи·Ҝеҫ„йҖүжӢ© е’Ң terminal stateпјҢйғҪжҳҜе’ҢеүҚж–ҮдёӯиҝҷдёӘи’ҷзү№еҚЎжҙӣж ‘жІЎжңүе…ізі»зҡ„пјҢзӣёеҪ“дәҺеңЁзҺ°еңЁиҝҷдёӘзҠ¶жҖҒдёӯеҸҲејҖдёҖеұҖжёёжҲҸпјҢйҖҡиҝҮдёҚж–ӯзҡ„йҡҸжңәж“ҚдҪңпјҢжқҘжЁЎжӢҹеҮәжёёжҲҸзӣҙеҲ°жёёжҲҸз»“жқҹпјҢз»ҷеҮәжҲ‘д»¬дёҖдёӘ value жү“еҲҶпјҒе’Ңд№ӢеүҚжҲ‘д»¬е»әз«Ӣзҡ„и’ҷзү№еҚЎжҙӣж ‘жҳҜжІЎжңүе…ізі»зҡ„пјҒеҰӮжһңиҜҙжңүе…ізі»пјҢйӮЈе°ұжҳҜrolloutзҡ„еҲқе§ӢзҠ¶жҖҒжҳҜеҹәдәҺи’ҷзү№еҚЎжҙӣж ‘зҡ„йӮЈдёӘиҠӮзӮ№зҡ„зҠ¶жҖҒгҖӮ

rolloutпјҲжЁЎжӢҹпјүйҳ¶ж®өзҡ„жүҖжңүзҠ¶жҖҒеҸҳеҢ–гҖҒйҡҸжңәйҖүжӢ©гҖҒзӣҙеҲ° terminal state зҡ„иҝҮзЁӢпјҢ
йғҪдёҚжҳҜи’ҷзү№еҚЎжҙӣжҗңзҙўж ‘пјҲMCTSпјүдёӯзҡ„иҠӮзӮ№гҖӮ

е®ғд»¬еҸӘжҳҜдёҖеңәвҖңеҹәдәҺеҪ“еүҚеҸ¶иҠӮзӮ№зҠ¶жҖҒзҡ„йҡҸжңәжёёжҲҸвҖқгҖӮ

е”ҜдёҖзҡ„иҒ”зі»жҳҜпјҡ
rollout зҡ„иө·зӮ№пјҲеҲқе§ӢзҠ¶жҖҒпјү= ж ‘дёӯеҲҡеҲҡжү©еұ•еҮәжқҘзҡ„йӮЈдёӘеҸ¶иҠӮзӮ№зҡ„зҠ¶жҖҒгҖӮ

жЁЎжӢҹзҡ„з»“жһңпјҲvalueпјүдјҡеҸҚеҗ‘дј ж’ӯеӣһж ‘йҮҢпјҢ
жӣҙж–°еҸ¶иҠӮзӮ№еҸҠе…¶зҘ–е…ҲиҠӮзӮ№зҡ„д»·еҖјдј°и®ЎгҖӮ

еҸҜд»Ҙиҝҷж ·зҗҶи§Јпјҡ

  • MCTS ж ‘зҡ„дҪңз”Ёпјҡдҝқеӯҳе·Із»ҸжҺўзҙўиҝҮзҡ„гҖҒиҫғжңүд»·еҖјзҡ„зҠ¶жҖҒз©әй—ҙпјҲвҖңең°еӣҫвҖқпјүгҖӮ
  • rollout зҡ„дҪңз”Ёпјҡд»ҺвҖңең°еӣҫиҫ№зјҳвҖқеҫҖеӨ–йҡҸжңәзңӢзңӢжңӘжқҘдјҡеҸ‘з”ҹд»Җд№ҲпјҲвҖңжҺўи·ҜвҖқпјүгҖӮ

еӣӣгҖҒback propagationпјҲеҸҚеҗ‘дј ж’ӯпјү

д»Һ rollout дёӯжүҫеҲ°йҖӮеҗҲзҡ„еҖјпјҢж”ҫеҲ°ж ‘дёӯйҖӮеҗҲзҡ„дҪҚзҪ®дёӯеҺ»гҖӮ

е°ұжҳҜжҠҠ value зҡ„еҖјд»Һи·Ҝеҫ„зҡ„иҠӮзӮ№дёӯеҠ дёҖйҒҚгҖӮ

з®—жі•з»Ҳз»“

1гҖҒйҷҗеҲ¶ж—¶й—ҙ

2гҖҒеӣәе®ҡиҝӯд»Јж¬Ўж•°

з»“жқҹж—¶йҖүжӢ©дёҖдёӘ value жӣҙеӨ§зҡ„иҠӮзӮ№е®ҢжҲҗеҶізӯ–гҖӮ

дҫӢеӯҗ

иҝҷйҮҢзҡ„ T е°ұжҳҜ selection з« иҠӮдёӯд»Ӣз»Қзҡ„ valueгҖӮ NжҳҜжҖ»жҺўзҙўж¬Ўж•°пјҢnжҳҜиҝҷдёӘиҠӮзӮ№зҡ„жҺўзҙўж¬Ўж•°гҖӮ

еҲқе§ӢзҠ¶жҖҒеҰӮдёӢпјҢж №иҠӮзӮ№жҳҜеҸ¶еӯҗиҠӮзӮ№пјӣ

еңЁexpansionеҮәдёӨдёӘиҠӮзӮ№еҗҺпјҢеҸ‘зҺ°niйғҪдёә0пјҢеҜјиҮҙUCB1дёӨдёӘжӢ“еұ•зҡ„иҠӮзӮ№йғҪжҳҜвҲһгҖӮжүҖд»ҘUCB1жҳҜзӣёзӯүзҡ„пјҢзҗҶи®әдёҠйҖүи°ҒйғҪдёҖж ·гҖӮ

жҳҜеҸ¶еӯҗиҠӮзӮ№еҗҢж—¶еҸҲжІЎжңүиў«жҺўзҙўиҝҮпјҢж №жҚ®еүҚйқўзҡ„жөҒзЁӢеӣҫпјҢзӣҙжҺҘе°ұиЎҢ rolloutи®Ўз®—гҖӮ

и®Ўз®—еҮәдәҶдёҖдёӘvalueеҖјпјҢдёә20 гҖӮ

еҸҚеҗ‘дј ж’ӯпјҢжҠҠи·Ҝеҫ„дёҠзҡ„иҠӮзӮ№зҡ„д»·еҖјйғҪж”№жҲҗ20пјӣN_0 = 1 дәҶпјҢеӣ дёәиў«жҺўзҙўиҝҮдёҖж¬ЎдәҶгҖӮ

жӯӨж—¶йҮҚж–° д»Һж №иҠӮзӮ№еҮәеҸ‘пјҢеҶҚ жү§иЎҢдёҠйқўзҡ„дёҖиҪ®ж“ҚдҪңпјҢеҸ‘зҺ° S_1 зҡ„UCB1е·Із»ҸжҳҜдёҖдёӘеӣәе®ҡеҖјдәҶпјҢиҖҢ S_2 з”ұдәҺ жІЎжңүиў«и®ҝй—®иҝҮпјҢиҝҷдёӘеҖјиҝҳжҳҜдёҖдёӘвҲһпјҢеҜјиҮҙдәҶжҲ‘д»¬дёҖе®ҡдјҡйҖүжӢ© S_2 иҠӮзӮ№гҖӮ

з»ҷ S_2 иҠӮзӮ№жқҘдёҖиҪ®д»ҘеҗҺпјҢеҒҮеҰӮ е®ғзҡ„еҖјжҳҜ 10пјҢеҲҷж №иҠӮзӮ№жҖ»и®ҝй—®ж¬Ўж•°дјҡз”ұдәҺеҸҚеҗ‘дј ж’ӯзҙҜеҠ пјҢеҸҳжҲҗ2пјҢvalueзҙҜеҠ жҲҗ20+10=30пјӣ

зҺ°еңЁејҖе§ӢйҮҚж–°д»ҺS_0иҠӮзӮ№ејҖе§Ӣз¬¬дәҢиҪ®гҖӮ

з®—еҮә UCB еҖјпјҢз»ҷ S_0 зҡ„жүҖжңүеӯҗиҠӮзӮ№жү“еҲҶгҖӮ

еҸ‘зҺ° S_1 иҠӮзӮ№еӨ§пјҢжүҖд»ҘйҖүжӢ© S_1 пјҢеҸ‘зҺ° S_1 жҳҜеҸ¶еӯҗиҠӮзӮ№пјҢдҪҶжҳҜе·Із»Ҹиў«и®ҝй—®иҝҮдәҶпјҢжҢүз…§жөҒзЁӢз»ҷе®ғ жһҡдёҫжүҖжңүеҸҜиғҪзҡ„еҠЁдҪңпјҢе№¶ж·»еҠ еҲ°ж ‘дёӯпјҢз„¶еҗҺ еҪ“еүҚиҠӮзӮ№ = з¬¬дёҖдёӘж–°иҠӮзӮ№пјҢз”ЁиҝҷдёӘиҠӮзӮ№еҺ»еҒҡ rollout и®Ўз®—гҖӮ

еҶҚback propagationпјӣ

иҝҷдёҖиҪ®жү§иЎҢе®ҢжҜ•д№ӢеҗҺеҸҲеӣһеҲ°дәҶ S_0 , жӯӨж—¶еҸҲйҮҚж–°ејҖе§ӢдёҖиҪ® selectionпјҢи®Ўз®— UCB пјҢдҪҶжҳҜйңҖиҰҒжіЁж„ҸпјҢиҝҷдёӘ Vi жҳҜе№іеқҮеҖјпјҢVi зҡ„еҖјйңҖиҰҒйҷӨд»–зҡ„жҺўзҙўж¬Ўж•°гҖӮ

жӯӨж—¶ S_2 зҡ„еҖјжӣҙеӨ§дёҖдәӣпјҢжүҖд»ҘпјҢжҲ‘д»¬зҺ°еңЁйҖүжӢ©зҡ„жҳҜ S_2 иҠӮзӮ№пјӣ

жҲ‘д»¬еҸҜд»ҘеҸ‘зҺ° S_2 жҳҜеҸ¶еӯҗиҠӮзӮ№пјҢеҸҲиў«жҺўзҙўиҝҮпјҢжҢүз…§жөҒзЁӢпјҢе…ҲеҜ№е®ғ жһҡдёҫжүҖжңүеҠЁдҪңеҠ е…Ҙж ‘дёӯпјҢз„¶еҗҺйҖүз¬¬дёҖдёӘеҠЁдҪңпјҢеҒҡrolloutи®Ўз®—пјҢеҶҚback propagationгҖӮ

еҒҮеҰӮжҲ‘д»¬зҡ„зЁӢеәҸеңЁжӯӨж—¶е°ұеҒңдёӢжқҘдәҶпјҢжҲ‘д»¬жқҘзңӢзңӢдјҡжҖҺд№Ҳжү§иЎҢпјҹ

з”ұдәҺжӯӨж—¶и®Ўз®— S_1 е’Ң S_2 зҡ„ UCB пјҢдјҡеҸ‘зҺ° S_2 зҡ„жӣҙеӨ§пјҢжүҖд»ҘдјҡйҖү еҸіиҫ№йӮЈжқЎи·Ҝеҫ„гҖӮ

жҖ»з»“

MCTSеғҸдёҖдёӘеёҰеҸҚеҗ‘дј ж’ӯе’Ңи·Ҝеҫ„иөӢеҖјеҠҹиғҪзҡ„дәҢеҸүж ‘пјҢдҪҶжҳҜйңҖиҰҒжҳҺзЎ®зҡ„дёҖзӮ№жҳҜпјҢдҪҶе®ғдёҚжҳҜеӣәе®ҡзҡ„гҖҢжҷ®йҖҡдәҢеҸүж ‘гҖҚпјҢиҖҢжҳҜж №жҚ®зӯ–з•ҘеҠЁжҖҒжү©еұ•зҡ„гҖҢйқһеӣәе®ҡеҲҶж”Ҝжҗңзҙўж ‘гҖҚгҖӮ

еңЁsimulationд»ҝзңҹдёӯпјҢз”ЁйҡҸжңәпјҲжҲ–еҗҜеҸ‘ејҸпјүж–№жі•вҖңзҺ©вҖқеҲ°жёёжҲҸз»“жқҹгҖӮ
рҹ‘ү иҝҷдёҖжӯҘе°ұжҳҜеә”з”ЁвҖңи’ҷзү№еҚЎжҙӣж–№жі•вҖқ вҖ”вҖ“ з”ЁйҡҸжңәж ·жң¬зҡ„з»ҹи®Ўз»“жһңеҺ»иҝ‘дјјдёҖдёӘж•°еӯҰй—®йўҳзҡ„зңҹе®һзӯ”жЎҲгҖӮ

MCTS зҡ„жҖқи·Ҝпјҡ

вҖңйҡҸжңәе°қиҜ•еӨҡж¬ЎпјҢз»ҹи®Ўе“ӘжқЎи·Ҝеҫ„иғңзҺҮй«ҳпјҢз„¶еҗҺдјҳе…ҲжҺўзҙўе®ғгҖӮвҖқ

еҸҜд»Ҙиҝҷж ·зҗҶи§Јпјҡ

  • MCTS ж ‘зҡ„дҪңз”Ёпјҡдҝқеӯҳе·Із»ҸжҺўзҙўиҝҮзҡ„гҖҒиҫғжңүд»·еҖјзҡ„зҠ¶жҖҒз©әй—ҙпјҲвҖңең°еӣҫвҖқпјүгҖӮ
  • rollout зҡ„дҪңз”Ёпјҡд»ҺвҖңең°еӣҫиҫ№зјҳвҖқеҫҖеӨ–йҡҸжңәзңӢзңӢжңӘжқҘдјҡеҸ‘з”ҹд»Җд№ҲпјҲвҖңжҺўи·ҜвҖқпјүгҖӮ

еҶҚйҖҡиҝҮвҖңжҺўи·Ҝз»“жһңвҖқеҸҚеҗ‘дј ж’ӯ еӣһеҺ»дҝ®жӯЈжқҘMCTSж ‘гҖӮ
